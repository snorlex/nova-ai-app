## Setup
1. Create a `.env` file with your OpenAI API key:
```
OPENAI_API_KEY=sk-xxxx
```

2. Install dependencies:
```
pip install -r requirements.txt
```

3. Run the app:
```
python main.py
```

Place your personal writing or memory files in:
```
data/user_texts/
```
